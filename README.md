# Hello Maui

## Setup

### Maui Workload

1. `dotnet workload install maui`
1. `dotnet tool install -g redth.net.MAUI.check`
1. `maui-check`

## Run

### Android

- `bash scripts/android/run/emulator.ba.sh`
- `bash scripts/android/run/dotnet.ba.sh`
